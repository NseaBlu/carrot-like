# Canvas 塔防 Demo（保卫萝卜风格）

浏览器原生 **Canvas 2D** 实现的轻量塔防演示：路径寻敌、波次刷怪、塔升级与射程、萝卜防守与多关卡切换。玩法与视觉致敬经典「保卫萝卜」，适合作为塔防逻辑与数据配置的练习工程。

## 功能概要

- **塔防核心**：点击空地建造瓶子炮塔，点击已有塔升级；子弹射击、敌人沿路径移动，漏怪扣生命。
- **关卡与波次**：关卡与路径由 `theme1-paths.json` 配置；支持 URL 参数 `?level=` 指定关卡索引。
- **通关流程**：非最后一关通关后出现对话框，确认后再进入下一关；也可选择本关重开。
- **辅助显示（默认关闭）**：顶栏「辅助」开关可切换逻辑区边框、格子线、可建格高亮、不可建格红叉、塔射程圈、路径折线与顶点；路径冰砖贴图为常态路面表现。
- **操作**：鼠标点击建造 / 选塔升级；键盘 **R** 或「重开」重置当前局。

## 运行方式

本项目使用 ES Module，需通过 **HTTP** 访问（直接双击打开本地文件可能因模块策略无法加载）。

任选其一：

```bash
# Python 3
python -m http.server 8080

# Node（需安装 npx）
npx serve .
```

浏览器打开 `http://localhost:8080`（端口以实际为准），进入 `index.html` 所在目录即可。

可选：`?paths=` 指定自定义路径 JSON 的 URL；`?level=` 指定关卡。

## 仓库结构（节选）

| 路径 | 说明 |
|------|------|
| `index.html` | 页面壳与 HUD、通关对话框、辅助按钮 |
| `js/main.js` | 主循环、输入、胜负与进关逻辑 |
| `js/sceneDraw.js` | 场景绘制（路径、塔、敌人、辅助格线等） |
| `theme1-paths.json` | Theme1 关卡路径与元数据 |
| `assets/` | 背景、萝卜与塔等图片资源 |
| `docs/` | 坐标导出说明与搭建备忘 |
